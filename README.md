## ğŸ“¥ Installation

### **Step 1:** Download the Extension
1. Go to the [GitHub repository](https://github.com/Farlapata/litbuytools)
2. Click the green **"Code"** button
3. Select **"Download ZIP"**
4. Extract the ZIP file to a folder on your computer

### **Step 2:** Open Chrome Extensions
1. Open Chrome and go to: `chrome://extensions/`
2. Turn on **"Developer mode"** (toggle in top-right corner)

### **Step 3:** Load the Extension
1. Click **"Load unpacked"**
2. Select the **`LitbuyTools`** folder you extracted
3. Done! You'll see the LitbuyTools icon in your toolbar

### **Step 4:** Start Using
Visit any Litbuy product page and enjoy the new features!
